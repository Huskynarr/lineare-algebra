# Sicherheitsrichtlinie

## Unterstützte Version

Sicherheitskorrekturen werden auf dem aktuellen Stand des `main`-Branches gepflegt. Ältere Commits und lokale Forks werden nicht separat unterstützt.

## Schwachstelle melden

Bitte veröffentliche Sicherheitslücken nicht als öffentliches Issue. Nutze stattdessen GitHubs private Meldung unter **Security → Report a vulnerability** oder öffne direkt einen [privaten Security Advisory](https://github.com/Huskynarr/lineare-algebra/security/advisories/new).

Beschreibe möglichst:

- betroffene URL oder Datei,
- nachvollziehbare Schritte zur Reproduktion,
- erwartete und tatsächliche Auswirkung,
- Browser und Betriebssystem,
- mögliche Abhilfe, falls bekannt.

Meldungen werden so zeitnah wie möglich geprüft. Nach einer bestätigten Behebung kann eine koordinierte Veröffentlichung vereinbart werden.

## Datenschutz

Die App benötigt kein Konto und sendet keinen Lernfortschritt an einen Server. Savegames liegen ausschließlich im `localStorage` des Browsers oder in einer vom Nutzer exportierten JSON-Datei.
